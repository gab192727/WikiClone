Privacy Policy

Last updated: October 2025

This website (the “Application”) is a student-demo project built with Django for educational purposes. We respect the privacy of all visitors.

Data Collection & Use

  • The Application does not collect, store, or share any personal information about its users (no registration, no analytics, no cookies for user tracking).
  • The only data fetched by the Application is publicly available content from third-party APIs: namely the Wikipedia and Wiktionary APIs.
  • Because no personal information is collected, encrypted, or transferred to external parties (other than the publicly accessible APIs), there are no user profiles or personal data warehouses.

Third-Party Services

  • The Application retrieves content from the Wikimedia Foundation’s projects (Wikipedia, Wiktionary) under their open content licenses.
  • These services have their own privacy and data practices. This Application does not assume responsibility for those practices.

Cookies & Tracking

  • The Application does not deploy cookies or tracking mechanisms for user identification. If future enhancements add such features, they will be disclosed here.

Security

  • Since no personal data is processed, the security risks associated with such data are not applicable. Nevertheless, standard security practices of the host environment apply (e.g., use of HTTPS, secure static file serving).

Changes to this Policy

  • We reserve the right to update this Privacy Policy at any time. When changes are made, the “Last updated” date will reflect the new version.
  • Continuing to use the Application after changes implies acceptance of the updated policy

If you have any questions about this Privacy Policy, you may contact the developer via the GitHub repository.
